# GitHub Pages Hologram Shirt Demo

## What to upload (repo root)
- `index.html`
- `shirt.png`
- `images/lt.png`
- `images/gt.png`

## Enable GitHub Pages
Repo -> **Settings** -> **Pages**
- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

## Your public URL
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

## Swap the hologram images later
Replace these files (keep the same names):
- `images/lt.png`
- `images/gt.png`
